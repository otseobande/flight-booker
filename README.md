## Features

- User registration
- User login

## Setup

- Install virtualenv

```bash
pip install virtualenv
```

- Create the application's virtual environment

```bash
virtualenv venv
```

- Activate the virtual environment

```bash
source venv/bin/activate
```

- Install dependencies

```bash
pip install -r requirements.txt
```
